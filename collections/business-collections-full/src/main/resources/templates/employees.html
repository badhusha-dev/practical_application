List Collection Characteristics:
  • ArrayList: Resizable array implementation, O(1) random access, O(n) insertion/deletion
  • LinkedList: Doubly-linked list implementation, O(n) random access, O(1) insertion/deletion
  • Use ArrayList when: Frequent random access, fewer insertions/deletions
  • Use LinkedList when: Frequent insertions/deletions, sequential access
